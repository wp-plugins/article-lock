A simple plugin that locks a public post, shows only its preview and unlocks the entire content on the desired date.

== Description ==

If you ever wanted to publish a post or article, but only show a preview of it and unlock the whole content on a desired date, than this is the plugin
for you.

What this plugin offers:

* Limit post preview to desirable length of characters
* Show number of images, code snippets and words that the whole post contains
* Customizable post preview length of each post
* Customize position of post preview message (top or bottom)
* Add HTML before or after post preview message

== Installation ==

1. Upload the plugin directory to to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize the settings in Settings->Article Lock
4. When adding a new post or editing an existing one, config the options in the sidebar 'Article Lock'.
